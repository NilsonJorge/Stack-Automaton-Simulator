	Foi utilizada a linguagem Python na versão 3.11.0.
	A Idle utilizada foi o PyCharm Community Edition 2022.3
	Os arquivos .txt(com as tabelas de transições e as strings que serão testadas)
devem estar no mesmo diretório do arquivo .py que contêm o código-fonte.
	É possível rodar o programa pelo CMD do Windows 10 ou Terminal do Linux 
acessando o diretório onde está o codigo fonte e utilizar o comando "python3 SimuladorPDA.py".
Para isso basta ter o python 3 instalado no sistema.
